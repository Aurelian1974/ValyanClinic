# Table Structure Documentation

This folder contains table creation scripts and schema definitions extracted from the ValyanMed database.

Each `.sql` file represents a table with its complete structure including:
- Column definitions
- Data types
- Constraints
- Primary keys
- Foreign keys
- Indexes

## File Naming Convention
`TableName_Structure.sql` - Complete table definition script
`TableName_Indexes.sql` - Index definitions for the table
